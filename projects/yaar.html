YAAR — Empowering Entrepreneurs & SHGs

A responsive static website promoting entrepreneurship, SHG products, MSME courses, and robotics learning programs.

Overview

YAAR is a lightweight, responsive website built to support entrepreneurs, students, MSMEs, and SHG communities. It highlights programs, showcases products, promotes training opportunities, and increases visibility for community-driven initiatives — while staying fast, simple, and easy to update.

Key Objectives

Mobile-First Website

Deliver a fast, responsive site built using HTML, CSS and Tailwind.

Program Visibility

Make entrepreneur training, MSME support, and robotics kits easy to explore.

Promote SHG Products

Support local sellers with simple marketplace-style pages.

Trust & Impact

Show testimonials, metrics, and initiatives to build credibility.

Clear CTAs

Encourage registrations, partnerships, and program inquiries.

Future-Ready

Structured for easy upgrades like CMS, payments, and multi-language.

Core Features

Home Page Highlights

Banner promos, impact counters, testimonials, and strong CTAs.

Entrepreneur Programs

Business model, finance literacy, marketing resources & forms.

SHG Marketplace

Category listings, seller highlights, and inquiry-based checkout.

MSME & Robotics

Course catalog, schedules, and robotics learning kit pages.

Tech Stack
